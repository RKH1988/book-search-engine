# Book Search Engine

## Description
This app uses Google Books API search engine built with a RESTful API to allow users to save book searches to the back end.

## Technologies Used
- MERN
- MongoDB
- Node.js/Express.js
- GraphQL with Apollo Server
- JavaScript
- React
- Mongoose
- Heroku
